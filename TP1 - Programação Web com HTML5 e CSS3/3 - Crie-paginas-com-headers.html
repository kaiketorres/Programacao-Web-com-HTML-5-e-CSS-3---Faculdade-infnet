Explicar como funciona a Web e o que é uma página web

A Web ou World Wide Web é um sistema de informação distribuído globalmente que fornece acesso a inúmeros documentos e outros recursos, como imagens e vídeos, interligados através da Internet. Consiste em milhões de sites e páginas da web criados por indivíduos, empresas, organizações e governos em todo o mundo.

Explicar o que são ambientes de programação?

Um ambiente de programação, ou ambiente de desenvolvimento integrado (IDE), é um conjunto de ferramentas e recursos integrados em um único software para facilitar o desenvolvimento, teste e depuração de programas de computador. Este ambiente fornece aos programadores uma interface unificada que combina muitos recursos importantes para tornar o processo de desenvolvimento de software mais eficiente e produtivo.
